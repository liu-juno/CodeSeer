# 页面风格重构设计方案

**日期**: 2026-06-19
**需求**: 重构整个项目的基础页面风格
**参考**: TAPD 腾讯敏捷协作平台风格

---

## 1. 设计目标

将现有"AI 化"紫/靛蓝渐变风格重构为腾讯 TAPD 的企业级蓝色系风格，包括：
- 整体配色系统
- 导航结构（图标边栏 + 顶部 Tab）
- 路由结构（扁平 → 嵌套）
- 组件样式（圆角、阴影、卡片）

---

## 4. 导航结构

### 侧边栏（宽 56px）
- 仅显示图标，hover 显示文字提示
- 图标大小：20px
- 背景：`#ffffff`，右边框：`1px solid #e8e9eb`

### 顶部 Tab 栏（高 48px）
- 显示当前模块名称和面包屑
- Tab 样式：文字 + 选中时下划线（主色）
- 背景：`#ffffff`，下边框：`1px solid #e8e9eb`

### 导航项
- 项目管理
- 迭代管理
- 需求管理
- (其他保留)

---

## 5. 路由结构

### 扁平结构 → 嵌套结构

| 当前 | 重构后 |
|------|--------|
| `/projects` | `/project/:id` |
| `/iterations` | `/project/:id/iterations` |
| `/requirements` | `/project/:id/requirements` |
| `/iterations/:id` | `/iteration/:id` |
| `/requirements/:id` | `/requirement/:id` |

### 嵌套路由示例
```
/project/:id              → 项目概览（含迭代/需求Tab）
/project/:id/iterations   → 迭代列表
/project/:id/requirements → 需求列表
/iteration/:id            → 迭代详情
/requirement/:id          → 需求详情
```

---

## 6. 组件样式

### 卡片组件
- 背景：`#ffffff`
- 圆角：12px
- 阴影：`0 2px 8px rgba(0,0,0,0.08)`
- 内边距：20px

### 表格组件
- 表头背景：`#f5f7ff`
- 表头文字：`#646a73`，字号 12px，大写
- 行高：52px
- 行 hover：`#fafafa`
- 边框：`1px solid #e8e9eb`

### 按钮
**主按钮**
- 背景：`#2d5bff`
- 文字：`#ffffff`
- hover：`#1e4ae8`

**次按钮**
- 背景：`#ffffff`
- 边框：`1px solid #2d5bff`
- 文字：`#2d5bff`

**幽灵按钮**
- 背景：透明
- 边框：`1px solid #e8e9eb`
- 文字：`#646a73`

### 状态徽章
| 状态 | 背景色 | 文字色 |
|------|--------|--------|
| 进行中 | `#e6f0ff` | `#2d5bff` |
| 已完成 | `#dff5eb` | `#00a870` |
| 已取消 | `#f5f5f5` | `#969ba4` |

---

## 7. 实施顺序

1. **全局样式** - `main.css` 配色、圆角、阴影系统
2. **侧边栏** - 改为图标模式，56px 宽度
3. **顶部 Tab** - 新增顶部 Tab 导航组件
4. **路由调整** - 改为嵌套路由结构
5. **Projects 页面** - 重构为 TAPD 风格
6. **Iterations 页面** - 重构为 TAPD 风格
7. **Requirements 页面** - 重构为 TAPD 风格

---

## 8. 验收标准

- [ ] 页面整体配色符合 TAPD 蓝色系风格
- [ ] 侧边栏显示为图标模式
- [ ] 顶部显示模块 Tab 和面包屑
- [ ] 路由结构变为嵌套形式
- [ ] 卡片圆角为 12px，带阴影
- [ ] Projects/Iterations/Requirements 页面样式统一

---

## 9. 不在范围内

- 后端 API 修改
- 业务逻辑变更
- 非优先页面的重构（Dashboard、Settings 等暂不修改）
