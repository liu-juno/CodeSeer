# Modules 页面重新设计

## 背景
Modules.vue 页面存在信息层级不清的问题：顶部4个统计卡片、模块详情卡、文档卡片、Skill卡片堆砌在一起，用户难以快速找到关键信息。

## 设计方案

### 整体布局
保持左侧模块树 + 右侧详情区的基本结构不变。

### 改动点

1. **移除顶部统计卡片**
   - 删除总文档、草稿、已归档、模块数 4个统计卡片
   - 相关统计数字整合到模块详情 header 中

2. **模块详情 header 改造**
   - 显示：模块名称、描述、路径
   - 增加统计数字：文档数量、子模块数量
   - 操作按钮：新建子模块、编辑、删除、创建 Skill

3. **内容区改为标签页**
   - 两个 tab：「归档文档」「Skill」
   - 用户点击切换，不同时堆砌显示

- `frontend/src/views/Modules.vue`
